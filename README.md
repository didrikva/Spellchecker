# Spellchecker
Ett webbaserat spellchecker-projekt byggt i Python med hjälp av en Trie-datastruktur.

## Clona repo: 
git clone https://github.com/didrikva/Spellchecker.git
cd Spellchecker

## Installation:
pip install flask

## Starta applikation:
python app.py

Öppna sedan http://localhost:5000

## Requirements
- Python 3.8+
- Flask
- Jinja2
